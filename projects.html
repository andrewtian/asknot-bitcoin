Ask not what Bitcoin can do for you, but what you can do for Bitcoin

Bitcoin's prosperity and success depends on supporters like you. Contributing to these open-source projects will help the maintaince and innovation of critical infrastructure powering Bitcoin.

Clients

Clients provide the end-user an easy way to use Bitcoin.

  • Bitcoin Core no-frills client

    • Reference client powering majority of the Bitcoin network.
    • Written in C++

  • Armory client for advanced users

    • Bitcoin client for advanced users; piggybacks ontop the Satoshi client
    • Advanced features allowing multiple-wallet management, coin control, offline transactions, and more.
    • Written in C++ and Python 2.7.

  • MultiBit lightweight and international client

    • Lightweight Bitcoin client based on client-server protocol.
    • Features multiple languages.
    • Written in Java.
    • Requires no blockchain download.

  • Electrum simplistic lightweight client

    • Lightweight Bitcoin client based on client-server protocol.
    • Keys can be recovered if local data is lost.
    • Requires no blockchain download.
    • Written in Python 2.7.

  • Bitcoin Wallet for Android android client

    • Based on BitcoinJ implemenation.
    • Used by over 50,000.
    • Most popular Bitcoin client for Android
    • Written in Java.

  • Hive Web web-based Bitcoin wallet

    • server and client components of a web-based wallet
    • SPV implmentation allows for a lightweight client
    • Written in Javascript

  • Hive iOS iOS Bitcoin client

    • PhoneGap version of Hive Web
    • Written in Javascript

  • Hive Android android client

    • Based on Bitcoin Wallet for Android
    • Offers additional support for QR codes and NFC
    • Written in Java

  • bitc thin SPV CLI client

    • SPV implmentation allows for a lightweight client
    • CLI interface
    • Written in C

Nodes

Full node implmenetations.

  • btcd Bitcoin node in Golang

    • Designed solely to be a node (excludes wallet functionality)
    • Written in Golang

  • BitcoinJ Java implementation of Bitcoin

    • Well documented and maintained
    • Modular
    • Features multiple modes for different use cases.
    • Written in Java

Pool Servers

  • Eloipool fast python3 poolserver

    • Internal work generation
    • Used by many popular pools
    • Written in Python 3

  • P2Pool decentralized Bitcoin mining

    • Fully-decentralized Bitcoin pool
    • Written in Python 2.7

Miners

  • bfgminer features improvements over cgminer

    • focused on ASIC/FPGA device support
    • Written in C

  • cgminer GPU/ASIC/FPGA miner

    • Written in C
    • fork of bfgminer

Libraries

Useful language-specific libraries to make your use of Bitcoin easier.

  • libbitcoin async C++ Bitcoin library

    • Asynchronous
    • Written in C++

  • python-bitcoinlib Python Bitcoin library

    • Python 3 support
    • Written in Python

  • BitcoinJ Java implementation of Bitcoin

    • Features a node implementation
    • Well documented and maintained
    • Modular
    • Written in Java

Utilities

Various Bitcoin utilities for advanced Bitcoin users.

  • Brainwallet javascript web app

    • Offline Bitcoin key generation
    • Generate keys based on rememberable phrases

  • sx Modular Bitcoin commandline utilities

    • Generate, sign, and broadcast Bitcoin transactions straight from the commandline
    • Uses libbitcoin
    • Written in C++
